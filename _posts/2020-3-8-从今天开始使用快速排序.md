但是作为一个合格的程序员，我们必须了解其他优秀的排序算法，特别是快速排序。

我们先说一下它的速度：

**最佳情况：T(n) = O(nlogn)   最差情况：T(n) = O(n2)   平均情况：T(n) = O(nlogn)**　

而选择排序永远是O(n2)，真稳定，emmmmmmmmmmmmmmmm

```java
    public static void myquickSort(int[] arr,int start,int end) {
    	if(start>end) {
    		return;
    	}
    	int i,j;
    	i=start;
    	j=end;
    	int p = arr[start];
    	while(i<j) {
    		while(arr[j]>=p && i<j) j--;
    		while(arr[i]<=p && i<j) i++;
    		if(i<j) {
    			arr[i] = arr[i]^arr[j];
    			arr[j] = arr[i]^arr[j];
    			arr[i] = arr[i]^arr[j];
    		}
    	}
    	arr[start] = arr[i];
    	arr[i] = p;
    	myquickSort(arr,start,j-1);
    	myquickSort(arr,j+1,end);
    	
    }
```

```javascript
function quickSort(arr,start,end){
    if(start>end){
        return;
    }
    let i,j;
    i=start;
    j=end;
    let p = arr[start];
    while(i<j){
        while(arr[j]>=p && i<j) j--;
        while(arr[i]<=p && i<j) i++;
        if(i<j){
            arr[i] = arr[i]^arr[j];
            arr[j] = arr[i]^arr[j];
            arr[i] = arr[i]^arr[j];
        }
    }
    arr[start] = arr[j];
    arr[j] = p;
    quickSort(arr,start,j-1);
    quickSort(arr,j+1,end);
}
```
